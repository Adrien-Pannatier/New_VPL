Block edition

How to delete a block

Blocks are stored in multiple files (ui.json/[programing_language].json/block-list.json/etc). Fortunately, a python script named block-edition.py is present in the UI folder to help you delete blocks.

To use the file, open a command prompt and type the following:

python block_edition.py delete_block "[Name]"

In case you need to see what are the names of the blocks used in the Stimuli-VPL, you can type:

python block_edition.py show_blocks

How to add a block

Similarly, block-edition.py can be used to add a template for blocks given a name 

python block_edition.py add_block

once the command is entered, the name and block entered, it is possible to compile and run the Stimuli-VPL interface. However, all files marked with a ✏️ still need implementation.

How to visually edit a block

The design of the blocks is stored in the Blocks.svg file. Vector graphic editing softwares like illustrator or inkscape can open this svg to edit the various elements inside.

Frequent issues

To design the Stimuli-VPL, some compatibility issues were encountered using illustrator. These issues will be described here, along with how to solve them.

  • XML line:
    • Problem: on top of the .svg file, a xml line appears indicating the encoding. This is not compatible with Stimuli-VPL interface.
      <?xml version="1.0" encoding="utf-8"?>
    • Solution: the line must be manually deleted before compilation.
  • Complex element issue:
    • Problem: complex elements in the .svg file such as spheres or text do not render correctly on the interface.
    • Solution: this issue is solved by transforming the elements into compound path or by expending the elements into shapes.

URL parameters

To access in depth properties, different elements can be added in the URL to modify the user interface. Here these elements will be listed

SyntaxAnswer possibilitiesRemarks
robotthymio-tdm/simsim is not used in this interface
wws://localhost:8597 (Thymio from Thymio suite)/ws://localhost:3000 (Thymio web bridge)The Stimuli-VPL interface was tested using Thymio suite
uilanguageen/de/fr/it
advmodetrue/falseNot used in this interface
languageaseba/js⚠️new buttons have not been implemented in javascript yet
uipath like “ui/svg/ui.json”can be modified in case multiple UI are to be used by the programs
blurgreyscale percentage (0-100)What even is that used for
greyscalegreyscale percentage (0-100)What even is that used for

To add parameters to the url, the parameter chain must start with the “?” delimiter. The different parameters fragments are separated with “#” delimiters.

Example of URL

https://adrien-pannatier.github.io/New_VPL/vpl_stimuli/index-svg.html?ui=ui/svg/ui.json&robot=thymio-tdm&language=aseba#w=ws://localhost:8597
