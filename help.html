Welcome to the Help Page

This page provides information on how to use the website effectively.

How to Use

On this website you can add select which version of the VPL you want to use. Simply click on the button representing the module you have to open the appropriate VPL

Add a new module folder

To add a new module folder, you need to add the module directory at the level of the index.html file. To learn how to create a module, please consult: help to edit the VPL

Add a new module button

To add a new module, you can click on the + button. You'll need to provide a name for the module, a URL and an image to display on the module button. Module button images need to be square to correctly show.
